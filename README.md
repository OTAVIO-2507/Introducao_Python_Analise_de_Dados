# 📝 Lista de Exercícios - Parte 1 - Parte 2 - Parte 3
Este projeto contém três partes de exercícios focados em fundamentos matemáticos. As listas de exercícios são estruturadas para ajudar no entendimento e prática dos conceitos para python, com foco em operações e problemas interpretativos.

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido utilizando as seguintes tecnologias e ferramentas:
- Jupyter Notebook: Para criação e execução dos exercícios de forma interativa.
- Python: Para cálculos e análise das soluções.
- Markdown: Para organização e explicação dos exercícios.

## 🚀 Como Começar
Baixe ou clone o repositório do projeto.
- Abra os arquivos Lista de Exercícios - Parte 1.ipynb e Lista de Exercícios - Parte 2.ipynb em seu Jupyter Notebook.
- Não é necessária configuração adicional para visualizar ou editar os exercícios.

## 📦 Como Funciona
A lista de exercícios está organizada em duas partes:
- Parte 1: Exercícios de operações básicas (adição, subtração, multiplicação e divisão) com foco em interpretação de problemas.
- Parte 2: Exercícios mais avançados que envolvem operações combinadas e problemas mais desafiadores.
Cada exercício vem com explicações, exemplos e a solução detalhada.

## 🔄 Funcionalidades
- Exercícios Interativos: Permite que os usuários testem suas soluções diretamente no Jupyter Notebook.
- Respostas Detalhadas: Soluções passo a passo para facilitar o entendimento dos conceitos abordados.
